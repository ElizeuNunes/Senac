##Aulas 1 - Variáveis

Resumindo, a variavel é como uma caixa, onde eu guardo um determinado valor, este valor podendo ser um numero, um texto, um valor lógico(sim/não), etc.

###Tipo de Variáveis
- **String** - textos
- **int** - números inteiros
- **double** - números decimais

```
void(){
  String nome = "Elizeu"
  int idade = 16;
  double altura = 1.75;
}
```
